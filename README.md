## About the project

This project will include various applications that will help obtain information about the availability of electricity and power outages.

- ActiveLight is a parser that processes incoming data and structures it into a more accessible form. Information is received via API from [Svitlobot](https://svitlobot.in.ua/index.html).
  Instructions for ActiveLight can be found [here](https://github.com/DiCode77/CheckLI/tree/main/ActiveLight#readme).
